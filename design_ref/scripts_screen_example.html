OAS 담당자가 추출/진단 스크립트의 파라미터를 입력하고, 실행 명령어를 확인한 뒤 실제 실행합니다. 1920x1200 화면에서 스크롤 없이 보이도록 압축한 최종 예시입니다.

Local Mode

환경 메타데이터 BAR 내보내기

선택한 작업의 실행 기준과 주의사항

exportarchive.sh
실행 위치$DOMAIN_HOME/bitools/bin

목적

서비스 인스턴스의 catalog, security, model 산출물을 BAR 파일로 추출해 백업 또는 이관에 사용합니다.

실행 구문 형식

exportarchive.sh <service instance key> <export directory> {optional parameters} < /path/exportpwd.txt

필수 파라미터

  • Service instance key
  • Export directory
  • Encryption password file path: password 한 줄만 저장된 서버 파일 경로

옵션 파라미터

  • includedata
  • nouserfolders
  • noconnectionparams
  • advancedoptions=<path>
  • encryptionpassword는 입력하지 않음(password file을 stdin으로 전달)

주의사항

  • 웹앱은 password 파일 내용을 읽지 않습니다. 사전에 준비한 password 파일 경로만 명령어에 포함합니다.
  • 실행 시 exportarchive.sh 명령이 < 파일경로 형태로 실행됩니다. 파일에는 password 한 줄만 저장하고 chmod 600 권한을 권장합니다.

실행 단계

입력 → 명령어 확인 → 실행

실제 실행 주의
1

파라미터 입력

필수 값과 선택 옵션을 입력한 뒤 실행될 명령어를 확인합니다.

입력 완료(명령어 확인)
2

명령어 확인 및 실행

쉘 스크립트와 입력 파라미터가 합쳐진 명령어를 확인한 뒤 실행합니다.
